# Elite Zone Apex Website

## Deploy to Vercel (Free)
1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → Import repo
3. No build settings needed for static HTML
4. Add custom domain: elitezoneapex.com in Vercel dashboard

## Deploy to Netlify (Free)
1. Drag and drop this folder to netlify.com/drop
2. OR connect GitHub repo for auto-deploy on push
3. Add custom domain in Netlify settings

## Custom Domain DNS
Point your domain registrar's nameservers to Vercel/Netlify or add A records as instructed in the dashboard.

## Local Development
Open `index.html` in your browser, or run a local server:
```bash
npx serve .
```
